# django-with-vuejs
FAST AND CLEAR IN DevOps.

本项目是最简版

### 第一步:安装前端依赖
```python
# 在 package.json 文件所在目录
npm install
```
### 第二步:构建前端
```
npm run build
```
### 第三步:通过Django 自带server启动项目
```python
# 在 manage.py 所在目录
python manage runserver
```
