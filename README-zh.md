# django-with-vuejs
FAST AND CLEAR IN DevOps.

[English README.md](/README-zh.md)

项目是最简版

--- 
### Step 1: 装前端依赖

在 package.json 文件所在目录
```
npm install
```

### Step 2: 构建前端

在 frontend 目录
```
npm run build
```

### Step 3: 通过Django 自带server启动项目

In the directory where manage.py is located
```
python manage runserver
```
